# Gallery Drift

一个用于副屏展示的图片轮播窗口。当前版本采用普通窗口方式运行，不再依赖 Electron，界面风格更简约、更轻，适合长期放在副屏上展示。

## 当前功能

- 普通窗口运行，启动更稳定
- 简约留白风格界面
- 图片自动轮播切换
- 手动上一张 / 下一张
- 一键进入/退出全屏
- 适合副屏持续展示

## 启动方式

### 双击启动

直接双击：`启动 Gallery Drift.bat`

### 命令行启动

```bash
npm install
npm run dev
```

默认地址：`http://localhost:4173`

## 下一步可扩展
